LOGISTIC REGRESSION – Titanic Survival Prediction

This assignment includes:

1. Logistic_Regression.ipynb – Model building and training
2. titanic_model.pkl – Saved trained model
3. Titanic_train.csv – Training dataset
4. Titanic_test.csv – Test dataset
5. app.py – Streamlit web application for prediction

How to Run:
1. Open terminal
2. Navigate to LOGISTIC REGRESSION folder
3. Run: streamlit run app.py

This project predicts whether a passenger survived the Titanic disaster using Logistic Regression.
